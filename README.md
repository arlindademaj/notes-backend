# Notes API

Simple REST API for managing notes built with **Node.js**, **Express**, and **MongoDB**.

## Features

- Create notes
- Get all notes
- Get note by ID
- Update notes
- Delete notes

## Tech Stack

- Node.js
- Express
- MongoDB
- Mongoose

## Installation

Clone the project

git clone https://github.com/arlindademaj/notes-backend.git

Install dependencies

npm install

Create a `.env` file

PORT=3000
MONGO_URI=your_mongodb_connection_string

Run the server

npm run dev

Server runs on  
http://localhost:3000

## API Routes

POST /notes  
GET /notes  
GET /notes/:id  
PUT /notes/:id  
DELETE /notes/:id
